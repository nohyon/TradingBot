# 알고리즘

이 폴더는 알고리즘으로 주식의 전망을 예측하는 프로그램들의 모음입니다.

한 종목을 넣으면 

1. 뉴스 
2. PER
3. 차트 

등을 비교하여 + - 점수를 부여합니다. 

만일 A000011종목이 뉴스+ PER - 차트 - 라면

A000011은 -1 의 점수를 갖게됩니다. 

다른 종목 A001011이 뉴스 + PER + 차트 + 라면

A001011은 3 의 점수를 갖게됩니다.

그러면 A01011은  A000011에 비해 큰점수를 가지므로, A01011종목을 추천합니다.

이 프로그램은 뉴스를 판별합니다.
